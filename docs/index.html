# Quick start with Docker Compose
docker-compose up -d

# Submit SMS via HTTP/2 API
curl -X POST http://localhost:8080/api/v1/sms \
  -H "Content-Type: application/json" \
  -d '{"destination": "+1234567890", "message": "Hello, 5G!"}'

Features

Everything you need for production SMS processing

Dual Protocol Support

SMPP 3.x/5.x for legacy systems and HTTP/2 REST API for modern 5G networks. One gateway for all your SMS needs.

5G Ready

HTTP/2 interface compatible with 3GPP TS 29.540 SMSF. Ready for next-generation mobile networks.

Observable

Prometheus metrics, Grafana dashboards, and health endpoints. Full visibility into your SMS pipeline.

Cloud Native

Docker images, Kubernetes manifests, and HPA for auto-scaling. Deploy anywhere with confidence.

High Performance

Async processing with configurable thread pools. Benchmarked with JMH for predictable performance.

Production Ready

Battle-tested with comprehensive test coverage. Used in production environments worldwide.
